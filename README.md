# 🛒 E-Commerce Backend API

## 🚀 Overview
A full-featured e-commerce backend system for managing products, orders, users, and payments.

---

## ✨ Features
- 🔐 JWT Authentication & Role-Based Authorization
- 📦 Product Management (CRUD)
- 🛒 Cart System
- 📑 Order Management
- 💳 Payment Integration (Stripe/PayPal ready)
- 👤 User Management
- 📊 Admin Dashboard APIs

---

## 🛠 Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- AutoMapper
- JWT Authentication

---

## 🧱 Architecture
- Clean Architecture
- Repository Pattern
- DTOs
- Service Layer

---

## ▶️ Getting Started
1. Clone repository
2. Set connection string
3. Run migrations
4. Start API
